# 🍽️ Food Ordering Behaviour and Consumer Trends: A Structured Analysis of Choices and Habits

## 📌 Project Overview

The **Food Ordering Behaviour and Consumer Trends: A Structured Analysis of Choices and Habits** is an interactive Tableau dashboard developed to analyze food ordering behavior. It transforms raw food delivery data into meaningful visual insights, helping businesses understand customer preferences, revenue performance, delivery efficiency, and ordering trends.

The dashboard enables users to explore the dataset through interactive filters and multiple visualizations, making business analysis simple and effective.

---

## 🎯 Project Objectives

- Analyze customer ordering behavior.
- Identify popular cuisines across different cities.
- Evaluate revenue generated by different meal types.
- Measure customer satisfaction using ratings.
- Monitor delivery performance.
- Support data-driven business decisions.

---

## 📊 Dashboard Features

- 📌 Key Performance Indicators (KPIs)
  - Total Orders
  - Total Order Value
  - Delivery Fee
  - Average Customer Rating

- 🌍 City vs Cuisine Heat Map
- 👥 Orders by Age Group & Restaurant Type
- 🍴 Company Type vs Meal Type Analysis
- ⭐ Customer Rating by Meal Type (Donut Chart)
- 💰 Revenue Contribution by Meal Type
- 🥧 Cuisine Preference Distribution
- 🏙️ Top 5 Cities by Order Volume
- 🚚 Delivery Time Distribution

---

The dashboard allows users to filter data based on:
